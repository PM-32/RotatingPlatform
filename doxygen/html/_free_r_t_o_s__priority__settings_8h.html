Подробное описание

используется для настройки приоритетов
прерываний для корректной работы FreeRTOS.

Внимание
Приоритет прерывания, в котором используется API FreeRTOS (например, xSemaphoreGiveFromISR),
должен быть ниже, чем configLIBRARY_MAX_SYSCALL_INTERRUPT_PRIORITY
(из файла FreeRTOSConfig.h) по смыслу (не численно!).
Приоритеты прерываний численно должны быть больше
configLIBRARY_MAX_SYSCALL_INTERRUPT_PRIORITY и
меньше или равно configLIBRARY_LOWEST_INTERRUPT_PRIORITY.
Это справедливо для Cortex-M3, где приоритет прерывания тем выше,
чем меньше численное значение приоритета (значение 0 = максимальный
приоритет прерывания), на других мк может быть иначе.
В данном случае имеется 16 уровней приоритетов, прерываниям
можно установить численные значения от 6 до 15.
